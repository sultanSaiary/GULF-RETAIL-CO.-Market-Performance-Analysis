# GULF-RETAIL-CO.-Market-Performance-Analysis

### Data Analyst 

---

## Project Overview

An end-to-end data analysis project for Gulf Retail Co., 
a mid-sized retail chain operating 45 stores across Saudi Arabia.

**Business Question:**
> "We have 12M SAR budgeted to open 5 new stores in Q3 2026. 
Where should we expand and why?"

---

## Recommendation Summary

Based on 18 months of operational data, the recommended cities for 
the 5 new stores are:

| # | City | Reason |
|---|------|--------|
| 1 | Jeddah | Lowest rent (55K) + Highest loyalty (224) |
| 2 | Riyadh | Highest renevue + lowest markting cost + high customer loyalty |
| 3 | Jizan | Best profit margin + Zero complaints |
| 4 | Najran | Highest revenue per store (1.546M) |
| 5 | Makkah | Highest lifetime spend (13.4M) |
